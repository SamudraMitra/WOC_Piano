How to play the Piano:
Z key plays G note
S key plays Ab note
X key plays A note
D key plays Bb note
C key plays B note
V key plays C note
G key plays Db note
B key plays D note
H key plays E note
N key plays F note
J key plays Gb note
M key plays G note(higher pitch)
, key plays Ab note(higher pitch)
L key plays A note(higher pitch)
. key plays Bb note(higher pitch)
P key plays B note(higher pitch)
/ key plays C note(higher pitch)

To record a song while you play it, u need to press the 'Start' button before playing and when you stop playing, press the 'Stop' button.By pressing the 'Play' button you can hear the recording you just played. To store the recording, click on the 'Store' button to store it locally. To make it visible under 'My Recordings' , click on 'Update' and refresh the page.
